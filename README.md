# Panel SAGC — Productos automático

Esta versión agrega al proyecto real:

- código automático `PRD-000001`;
- categorías y marcas tomadas de los productos registrados;
- opción para crear una categoría o marca nueva desde el formulario;
- selección de imagen desde la PC;
- subida automática a una carpeta de Google Drive;
- generación del enlace `thumbnail`;
- guardado del enlace en la columna `IMAGEN` de `PRODUCTOS`;
- integración con Productos, Variantes, Mayoreo y Colores.

Consulta `APPS-SCRIPT/INSTRUCCIONES.txt` antes de implementar.
