# PANEL SAGC — PRODUCTOS V1

Primera versión funcional del núcleo del Panel SAGC.

## Incluye
- Sidebar contraíble en PC y ocultable en celular.
- Estado del sidebar recordado con localStorage.
- Módulo PRODUCTOS basado en las columnas reales:
  CODIGO, NOMBRE, MARCA, CATEGORIA, PRECIO, DESCUENTO,
  IMAGEN, IMAGEN_2...IMAGEN_5, VIDEO, DESCRIPCION, DESTACADO, ESTADO.
- Lista profesional de productos.
- Buscador por código, nombre, marca y categoría.
- Filtros por categoría y estado.
- Paginación.
- Nuevo producto.
- Edición.
- Activar/desactivar.
- Formulario lateral responsive.
- Datos demostrativos tomados de la estructura actual.
- Guardado local de demostración en el navegador.

## Importante
Esta versión todavía NO escribe en Google Sheets.
La siguiente etapa será crear la API segura de Apps Script y conectar:
PRODUCTOS, VARIANTES, MAYOREO, COLORES y ENLACES.
