# PANEL SAGC - PRODUCTOS V1.0

Módulo funcional para administrar productos, variantes, mayoreo y colores.

## Uso local
Abra `index.html` en Chrome.

## Conexión real
1. Instale los archivos de `APPS-SCRIPT` en el Apps Script ligado a su Google Sheets.
2. Implemente como aplicación web.
3. En el panel abra `Configuración` o pulse `Conexión`.
4. Pegue la URL `/exec`, guárdela y pruebe la conexión.
5. Pulse `Sincronizar`.

Esta versión no carga los tres productos de demostración. Al conectarse, muestra los registros reales de la hoja PRODUCTOS.
