# PANEL SAGC — Maqueta visual

Esta entrega contiene la estructura visual navegable del Panel SAGC.

## Incluye
- Dashboard
- Productos
- Categorías
- Marcas
- Inventario
- Ventas
- Compras
- Clientes
- Proveedores
- Caja
- Reportes
- Usuarios y roles
- Auditoría
- Configuración
- Diseño responsive para PC y celular

## Cómo abrir
1. Descomprime el ZIP.
2. Abre `index.html` en Chrome.

## Cómo publicar
Sube todos los archivos a un repositorio de GitHub y conecta ese repositorio a Cloudflare Pages.

## Estado
Es una maqueta visual. Todavía no está conectada a Google Sheets, Apps Script ni Firebase.
