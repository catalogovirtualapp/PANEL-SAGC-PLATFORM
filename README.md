# Panel SAGC — Productos V3

Versión local funcional del módulo PRODUCTOS.

## Incluye
- Diseño compacto y profesional.
- Pestañas Productos, Variantes y Mayoreo con contadores.
- Buscador inteligente sin pérdida de foco.
- Filtros, paginación y tabla responsive.
- Crear, editar, duplicar, activar, inactivar y eliminar productos.
- Variantes y mayoreo dentro de la ficha del producto.
- Validaciones y respaldo técnico JSON dentro del menú «Más».
- Persistencia local mediante localStorage.

## Ejecutar
Abra `index.html` en Chrome o Edge.

## Importante
Esta versión guarda datos en el navegador. La conexión multiusuario real con Google Sheets/Apps Script se implementa en la etapa de integración del backend.
