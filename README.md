# PANEL SAGC — PRODUCTOS V2

Versión funcional local del módulo PRODUCTOS.

Incluye:
- Buscador inteligente por palabras parciales y en cualquier orden.
- Corrección definitiva de pérdida de foco al escribir.
- Filtros por categoría, estado, variantes y mayoreo.
- Crear, editar, duplicar, activar/inactivar y eliminar productos.
- Variantes con opción 1, opción 2, imagen, stock y estado.
- Mayoreo por tramos con validaciones.
- Validación de códigos y combinaciones duplicadas.
- Exportación e importación JSON.
- Persistencia en localStorage.

Abrir `index.html` en Chrome o Edge.

Nota: esta entrega funciona 100% como panel local. La conexión multiusuario real con Google Sheets/Apps Script será la siguiente capa; no debe simularse con localStorage porque requiere servidor, bloqueos, autenticación y auditoría.
