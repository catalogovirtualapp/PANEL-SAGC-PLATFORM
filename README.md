# Panel SAGC — Productos V4

Abrir `index.html` en el navegador.

## Cambios V4

- Botón de contraer/expandir menú lateral funcionando en PC.
- El estado del menú queda guardado en el navegador.
- Módulo Configuración > Colores predeterminados.
- Crear y eliminar colores con nombre y selector hexadecimal.
- Vista previa en vivo del color.
- Los colores se enlazan automáticamente con Productos > Variantes.
- Las variantes muestran selector y muestra visual del color.
- Datos de demostración persistentes mediante localStorage.

## Nota

Esta versión permite probar el flujo real dentro del navegador. La siguiente integración será Google Sheets + Apps Script para compartir los datos entre varias computadoras y celulares.
