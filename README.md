# Panel SAGC · Productos Enterprise V2

Módulo profesional de Productos con:

- Código automático.
- Categorías y marcas reutilizadas desde los datos registrados.
- Productos, variantes y mayoreo.
- Colores predeterminados.
- Conexión con Google Sheets mediante Apps Script.
- Configuración de Google Drive desde el panel, sin editar código.
- Creación automática de carpeta de imágenes.
- Subida de imágenes desde la PC y guardado automático del enlace.

Consulta `APPS-SCRIPT/INSTRUCCIONES.txt`.
